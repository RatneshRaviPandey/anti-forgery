# OG Image Specifications — Infometa Technologies

## Dimensions
- **Standard:** 1200 × 630 pixels (Facebook, LinkedIn, Twitter summary_large_image)
- **Format:** JPG (for wide compatibility), WebP as fallback
- **Max file size:** 300KB

## Template Guide

### Home Page OG
- **File:** `/images/og/og-home.jpg`
- **Content:** Logo + tagline "Scan. Verify. Trust." + shield icon + gradient background (#0F766E → #115E59)
- **Text:** White, Inter Bold, 48px title, 24px subtitle

### Blog Article OG
- **Files:** `/images/og/blog-*.jpg`
- **Content:** Article title (2-3 lines max), category badge, "Infometa Blog" header
- **Text:** White title on dark teal overlay, 36px Inter Semibold, 18px category

### Industry Page OG
- **Files:** `/images/og/industry-*.jpg`
- **Content:** Industry icon (emoji or icon), industry name, tagline
- **Text:** 40px title, 20px subtitle, teal gradient background

### Case Study OG
- **Files:** `/images/og/case-study-*.jpg`
- **Content:** Case study title, key metric callout, company type
- **Text:** 36px title, 48px metric highlight, dark overlay

## Color Palette for OG Images
- Primary background: `#0F766E` → `#115E59` gradient
- Text: `#FFFFFF`
- Accent: `#ECFEFF`
- Badge: `#16A34A` (success), `#F59E0B` (warning)

## Typography
- **Font:** Inter (Google Fonts)
- **Weights:** 600 (Semibold), 700 (Bold)
- **Ensure text contrast ratio ≥ 4.5:1 against background**
